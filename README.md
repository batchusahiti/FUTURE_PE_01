# FUTURE_PE_01
## Velvet Brews Café Website

This project is a premium AI-generated café website created for the Future Interns Prompt Engineering Internship. The website combines cozy café aesthetics with modern UI design and high-conversion website copy.

---

### 🛠️ Core Features Developed
- Premium Hero Section
- Signature Brews & Menu Showcase
- Bakery & Pastries Gallery
- Custom Orders Section
- Cozy Café Corner
- Customer Testimonials
- Booking & Order CTA
- Responsive Modern UI Design
- Glassmorphism & Soft Shadow Cards
- Instagram-style Café Presentation

---

### 🎨 Design Style
- Modern Cozy Café
- Soft Earthy Theme
- Warm Cozy Lighting
- Elegant Typography
- Smooth Hover Effects
- Mobile-Friendly Layout

---

### 🎨 Color Palette
- Coffee Brown
- Cream White
- Caramel Gold
- Mocha Beige
- Warm Peach
- Soft Bronze Accents

---

### 🧠 Developed Prompt Logic Framework
- Structured prompts for generating homepage, services, and CTA content
- Iterative refinement of outputs for clarity and conversion
- AI-assisted design choices for layout, typography, and color palette
- Documentation of prompt engineering process in `prompt/`

---

### 📂 Repository Structure
- `prompt/` → Original prompts used
- `outputs/` → Generated text outputs (homepage, services, CTA)
- `screenshots/` → Screenshots of the published site
- `README.md` → Project documentation
